Output to standard output or log file

STAMP now keeps fairly quiet during its running, updating the user only after a pairwise/treewise/scan comparison has been compeleted. You can get lots of other output by using the -V (verbose) option. If you want a lot of output to be written to a file instead of the standard output, you can use -V in conjunction with -logfile file name.
